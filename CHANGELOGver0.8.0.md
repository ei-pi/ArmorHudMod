# Changelog

## [0.8.0] – Update

### English

- **Bug Fixes:** None.
- **Changes:**
    - The Armor HUD no longer leaves gaps and now automatically shifts itself.
    - Switched from Yarn mappings to Mojang mappings.
- **New Features:**
    - Added a new texture mode.
    - It is now possible to use the hotbar texture for the Armor HUD.
    - Buttons in the config menu now include tooltips with explanations.

---
